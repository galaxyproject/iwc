## Contiging Solo:

Generate assembly based on PacBio Hifi Reads.


### Inputs

1. Species Name [text]
2. Assembly Name [text]
3. Hifi long reads [fastq collection]
4. K-mer database [meryldb]
5. Genome profile summary generated by Genomescope [txt]
6. Bits for Hifiasm bloom filter [integer]
7. Homozygous Read Coverage. Optional, use if you think the estimation from Genomescope is inaccurate [integer]
8. Genomescope Model Parameters generated by Genomescope [tabular]
9. Database for Compleasm lineage (recommended: latest)
10. Compleasm lineage (recommended: vertebrata)
11. Name of primary assembly
12. Name of alternate assembly


### Outputs

1. Primary assembly
2. Alternate assembly
3. QC: Bandage image for the raw unitigs
4. QC: Compleasm report for both assemblies
5. QC: Merqury report for both assemblies
6. QC: Assembly statistics for both assemblies
7. QC: Nx and Size plots for both assemblies
